Phase 25 - UpdateSubscription (event subscription filter update, Domain + Node notifiers)
Live evidence against the demo stand (host <demo-host>, creds <redacted>).
Exercised end-to-end THROUGH the shipped MCP tool
(axxon_mcp_admin.update_event_subscription). UpdateSubscription only targets a
live PullEvents stream, so the tool opens a short-lived subscription on a
background daemon thread, applies UpdateSubscription with new filters, then
disconnects. The subscription is fully torn down; nothing persists.
