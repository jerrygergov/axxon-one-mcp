[change_lists_stream] ChangeListsStream (bidirectional stream, EPS_LAST on last packet)
  LIST    lists before = 0
  ADD     stream packet {added_list: id=<uuid> name=axxon-mcp-probe type=ELT_LPR, status=EPS_LAST}
          -> status=applied failed_lists=[] count 0->1
  REMOVE  stream packet {removed_lists: ids=[<uuid>], status=EPS_LAST}
          -> status=applied failed_lists=[] count 1->0 (restored)
