[perform_unit_action] reversible ARM -> DISARM round-trip (AcfaService.PerformAction)
  captured state:        ['DISARM']
  ARM    -> status=applied error_message=''
  state after ARM:       ['ARM']
  DISARM -> status=applied   (restore to captured state)
  final state:           ['DISARM']   (unit returned to its original state)

[vmda_cleanup] VMDAService.Cleanup on an empty-analytics camera
  vmda intervals before cleanup: 0  (verified empty via vmda_query, 24h window)
  vmda_cleanup -> status=applied result=True database=hosts/Server/VMDA_DB.0/Database
  (real RPC against the real VMDA db + a real, parseable camera id; the camera had no
   analytics, so nothing real was deleted)

[wire shapes] (probed before building)
  AcfaService.PerformAction(uid, id, [PropertyDescriptor]) -> {error_message, properties}.
    Empty error_message = success. uid = unit access_point.
  VMDAService.Cleanup(access_point=VMDA db, cs_IDs={camera_ID, schema_ID="vmda_schema"})
    -> {bool result}. camera_ID is the relative camera id (hosts/Server/ stripped).
